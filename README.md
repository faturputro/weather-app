# weather-app

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### App Configuration
- Token from Mapbox to render a WebGL Map
```
VUE_APP_MAPBOX_TOKEN=
```

- Weather API token
```
VUE_APP_API_TOKEN=
```

- Base URL of the weather app API Endpoint
```
VUE_APP_BASE_URL=
```
